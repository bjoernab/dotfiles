## 📦 Overview

This repository contains my personal configuration for:

- Hyprland (Wayland compositor)
- Waybar (status bar)
- Wofi / launcher
- Kitty (terminal)
- Mako (notifications)
- Shell (bash/zsh)

The goal is:
> ⚡ Fast setup • 🧼 Clean structure • 🔁 Easy portability
